# Môj plot widget – v19 clean balíček

Obsah balíčka:
- `widgetmp_mojplot_v19_fotky.html` – hlavný HTML súbor
- `fotky/` – priečinok s obrázkami používanými vo widgete
- `README_v19.md` – stručný popis balíčka

Zmeny vo verzii v19:
- v ZIPe je už iba jeden HTML súbor
- odstránené duplicitné README súbory
- ponechaný iba jeden aktuálny README
- nahradený obrázok `mojplot-podhrabova-doska.webp` realistickejšou verziou z produktovej stránky Môj plot

Poznámka:
Ak HTML súbor presúvate mimo ZIP balíčka, zachovajte priečinok `fotky/` vedľa HTML, aby sa obrázky načítali správne.
